# 💖 Website Xin Lỗi Ngọt Ngào 💖

Website xin lỗi đặc biệt với hiệu ứng 3D, animations và thiết kế pastel dễ thương.

## ✨ Tính năng nổi bật

- 🎨 **Thiết kế Pastel đẹp mắt**: Màu hồng nhạt - kem - trắng
- 💝 **Trái tim 3D tương tác**: Sử dụng Three.js với hiệu ứng xoay
- 🎵 **Nhạc nền lãng mạn**: Tự động phát khi tương tác
- 🌸 **Hiệu ứng visual**: Mưa trái tim, hoa bay, animations mượt mà
- 📱 **Responsive**: Hoạt động tốt trên cả mobile và desktop
- 🎮 **Tương tác vui nhộn**: Nút "giận tiếp" chạy trốn, popup dỗ ngọt
